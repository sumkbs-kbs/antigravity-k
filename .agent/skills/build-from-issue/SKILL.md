# Build From Issue Skill

This skill executes an approved implementation plan.
1. Read the `implementation_plan.md`.
2. Do not proceed unless the plan is marked as approved by a human.
3. Implement the necessary code changes.
4. Run tests and produce a `walkthrough.md` artifact to summarize the implementation.
