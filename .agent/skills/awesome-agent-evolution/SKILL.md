# Awesome Agent Evolution

You are an expert guide to the **Awesome Agent Evolution** knowledge base. This repository is a curated taxonomy of AI Agent self-evolution, memory systems, autonomous self-improvement, and the infrastructure that powers them.

## Usage Guidelines

When a user asks questions about "how to give an agent memory", "what is the best framework for autonomous coding", or "what are the latest papers on agent self-evolution", rely on this skill to consult the taxonomy.

## Core Taxonomies

1. **Evolution & Self-Improvement**
   - Frameworks enabling agents to mutate, learn, and rewrite code (e.g., Eliza, SuperAGI, EvoMap/evolver, Darwin Godel Machine).
2. **Memory Systems**
   - Vector, graph, episodic, and hybrid memory architectures (e.g., Mem0, Letta, Cognee, GBrain, TeleMem).
3. **Agent-to-Agent (A2A) & MCP Protocols**
   - Standards for inter-agent and host-agent communication (e.g., Google A2A, mcp-use).
4. **Agent Development Platforms**
   - Orchestration systems and visual tools for workflows (e.g., Dify, LangChain, OpenHands, PydanticAI).
5. **Agent Coding & Software Engineering**
   - Autonomous debugging and coding IDE assistants (e.g., Claude Code, Codex, SWE-agent, Roo Code).
6. **Embodied AI**
   - Connecting agents to physical or virtual robotics and interfaces (e.g., Open-AutoGLM, LeRobot, Nanobrowser).
