# Create Spike Skill

This skill is used when encountering unknown architectural challenges.
1. Do not write implementation code.
2. Investigate the codebase, frameworks, and tools required.
3. Produce an `implementation_plan.md` artifact detailing how the feature/fix should be built.
4. Set `request_feedback` to true and wait for human approval.
